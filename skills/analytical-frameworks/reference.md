# Analytical frameworks — how to run each

For every framework: gather evidence per cell, then step back and ask **"so what for the decision?"** The
grid is the input to the insight, not the output.

## SWOT
Strengths / Weaknesses (internal) × Opportunities / Threats (external). Fast and coarse. **Pitfall:** a list
with no prioritization or action. Force a "therefore we should…" from the top item in each quadrant.

## VRIO
For a candidate advantage, ask: **V**aluable? **R**are? costly to **I**mitate? **O**rganized to exploit? Only
yes-to-all is a *durable* advantage. Separates real moats from nice-to-haves.

## Porter's Five Forces
Rate each force (low/med/high) with evidence: (1) competitive rivalry, (2) supplier power, (3) buyer power,
(4) threat of substitutes, (5) threat of new entrants. High forces → thin profits. **Pitfall:** describing
forces without concluding whether the industry is attractive *and where you can shift a force in your favor*.

## Value chain
Map primary + support activities; find where value is added and where cost/margin leaks. Use to locate
differentiation and cost advantage.

## BCG growth-share matrix
Plot offerings on market growth × relative share: Stars, Cash Cows, Question Marks, Dogs. Guides
invest/hold/harvest/divest. **Pitfall:** treating share as the only value axis.

## Jobs-to-be-Done
"When [situation], I want to [motivation], so I can [outcome]." Focuses on the customer's progress, not your
features — surfaces non-obvious competitors (the job, not the category).

## PESTEL
Political, Economic, Social, Technological, Environmental, Legal macro forces. Use for market-entry and
long-horizon calls; keep to forces that actually move *this* decision.

## Anti-patterns (all frameworks)
- Filling the template but never stating the implication.
- Confusing description with analysis.
- Cherry-picking evidence to fit a predetermined conclusion (see ethics in `competitive-intel`).
